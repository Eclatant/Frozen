# 토지이용계획
- 토지수요예측 : x명을 수용하려면 땅이 얼마나 필요할지 예측
- 용도배분

땅을 용도에 맞게 구분한 후 수요예측 -> 배분  
용도지역지구설정 -> 필요하다면 지구단위계획

목표에 좋은 얘기 많이 나옴

## 계획수립과정
- 현황분석 및 장래 예측
- 계획목표 설정

## 토지 용도의 구분
크게 보면 주택, 일반 건축, 공원녹지, 도로  
일반 건축 : 상업, 교육, 사회문화복지

## 개발 밀도
단위 면적당 발생하는 활동의 강도  
뉴욕, 홍콩, 맨해튼 <-> LA  
하튼 적절한 밀도 제공해주는게 중요함

일반적으로 순밀도, 정밀도  
밀도 = 인구/면적 -> 순밀 정밀은 이 면적을 어떻게 계산하느냐의 차이

지하는 법적으로 용적률 안들어감

## 용도별 면적산정 사례
막 1만명 10만명 수용하는 도시 계획하라 하면 산출을 해야하는데  
사례를 통해 보는게 이해가 빠름  
요즘 추세는 인구밀도(인/ha)를 낮추는거, 판교는 100근처  
주택이 대략 40~50% 공원도 20% 공공시설도 20%

오른쪽은 너무크고 왼쪽의 한 대규모 단지 정도

### 시험 대비 : 거주밀도에 의한 면적 추정
택지 면적 산정  
혼합률이라는 개념 : 사람이 모두 주거지역에 살진 않음 상가건물에서 산다던가 하는 사람들을 10%라고 가정  
> 계획인구 30만, 혼합률 10%, 고밀:중밀:저밀 = 4:3:3, 각각 200, 150, 100명/ha일때 면적 추정  
고밀주거 : 30만 * 0.4 = 12만명 -> 12만명 / 200명/ha = 600ha -> 600ha * 0.9(혼합률) = 540ha

> 계획인구 20만, 혼합률 5%(다른 사례보거나 저변 계산해보거나),  
고밀:중밀:저밀(사업성 고려해보거나 아파트 너무많으니 중층고밀해보자거나) = 3:5:2  
답 : 고밀 285ha, 중밀 633ha, 저밀 380ha

### 주택이 몇 개 필요한가
계획인구에서 가구당 가구원 수로 사는지로 나누면 됨  
세대당 면적 : 단독 $200m^2$, 공동주택 $80m^2$  
또 혼합률 : 여기서는 전체분의 상업, 공업의 비율  
총 주택용지 : 순주택용지 / (1-공공공지율)  
주거용지면적 : 총주택용지 / (1-혼합률)

> 구해보면 157,416인, 세대당 인구 4인, 단독 면적 200, 공동주택 80, 단독:공동 = 5:5, 공공공지율 35%, 혼합률 20%  
총 세대 수 : 39,354세대(157416 / 4)  
순수한 단독 주택 용지 19677 * 200 = ㅁㄴㅇㄹm2  
거기에 도로랑 공지 붙이려면 ㅁㄴㅇㄹ / (1-0.35) = asdfm2  
asdf / (1-0.2) = 7568077m2 = 757ha  
공동주택용지도 비슷한 방법으로 = 3027230m2 = 303ha

> 해보자, 인구 10만, 가구원수 2.5명, 공동(80):단독(200) = 6:4, 공공공지율 30%, 혼합률 25%  
공동주택용지 365ha 단독주택용지 609ha = 974ha

### 상업업무용지
계획기준  
1인당 연면적 : 도심 12m2, 부도심 지역중심, 지구중심 : 10m2  
이쪽은 좀 대충함

산업지 이용인구에 의한 면적 추정  
상업지 면적 = 상업지이용인구 * 1인당 연면적  
총상업용지면적 = 상업지면적 / (1-공공용지율) <- 건물의 면적  
상업용지면적 = 총상업용지면적 / (층수*건폐율) <- 이게 곧 용적율 <- 나눴으니 얘는 땅의 면적

> 40만명, 상업지역이용인구 30%, 공공공지율 40%, 건폐율 60% 상업지층수 3층, 1인당 12m2  
계산하면 133ha
